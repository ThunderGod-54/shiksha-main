# ShikshaPlus

ShikshaPlus is an AI-powered learning platform built to make online education **focused, engaging, and accountable**.
Instead of measuring only course completion, ShikshaPlus tracks **real learner attention**, provides **adaptive AI guidance**, and delivers **verifiable learning outcomes**.

Live Demo:
[https://shikshaplus-gamma.vercel.app/](https://shikshaplus-gamma.vercel.app/)

## Why ShikshaPlus?

Most online learning platforms focus on content delivery and completion metrics.
ShikshaPlus addresses the **real problem in online education** — lack of focus, engagement, and measurable understanding.

By combining real-time attention tracking, AI-powered learning support, structured microlearning, and QR-verified certificates, ShikshaPlus shifts online learning from **passive consumption to engagement-driven outcomes**.

## What Makes It Different?

• Measures **attention**, not just completion
• Provides **instant AI guidance** when learners struggle
• Integrates learning, productivity, and notes in one platform
• Ensures **accountability** through QR-verified certificates
• Built using **Google’s secure and AI-first technologies**

## Core Features

• Real-time focus tracking
• AI-adaptive guidance (Gemini API)
• Structured microlearning modules
• Interactive quizzes
• Productivity tools and smart notes
• QR-verified downloadable certificates

## Tech Stack

Frontend: React.js with Vite
Backend: Flask and Firebase
AI Integration: Gemini API and NotebookLM
Storage: Firebase and Google Drive
Hosting: Vercel
Version Control: Git/ Github

## Google Technologies Used

Firebase – Secure authentication and backend services
Gemini API – AI-powered adaptive learning engine
Google Drive – Centralized note storage and access
NotebookLM – AI-powered research assistant

Built on Google’s secure and AI-first ecosystem.

## Project Summary

ShikshaPlus is designed around a simple but powerful idea:
**Online learning should measure engagement, not just completion.**

While existing platforms deliver content, ShikshaPlus ensures that learners stay focused, receive instant support when confused, and leave with verifiable proof of learning.
By leveraging real-time attention tracking and Google’s AI ecosystem, ShikshaPlus presents a scalable, practical, and impact-driven solution for modern online education.


## Getting Started

Clone the repository:

git clone [https://github.com/ThunderGod-54/shiksha-main.git](https://github.com/ThunderGod-54/shiksha-main.git)

Navigate to the project folder:

cd shiksha-main

Install dependencies:

npm install

Run the development server:

npm run dev
